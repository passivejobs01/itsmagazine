AMD RX 7600 Proxmox GPU PassThrough 설정 순서도
STEP 1. 그래픽카드 상태 체크
• GPU PCI 주소 확인: lspci | grep -i vga
• PCI ID 확인: lspci -nns 6e:00
• IOMMU 그룹 확인: find /sys/kernel/iommu_groups/ -type l | grep "6e:00"
↓
STEP 2. GRUB 설정 수정
• 현재 설정 확인: cat /etc/default/grub | grep CMDLINE
• amd_iommu=on iommu=pt 추가 (sed 명령어 사용)
• 적용: update-grub
↓
STEP 3. vfio 모듈 설정
• /etc/modprobe.d/vfio.conf 생성
• options vfio-pci ids=1002:7480,1002:ab30 추가
• softdep amdgpu pre: vfio-pci 추가
※ PCI ID는 그래픽카드마다 다를 수 있음
↓
STEP 4. initramfs 업데이트
• /etc/modules-load.d/vfio.conf 에 모듈 추가
• vfio / vfio_iommu_type1 / vfio_pci / vfio_virqfd
• update-initramfs -u -k all
↓
🔄 시스템 재부팅 (reboot)
↓
STEP 5. 부팅 후 정상 인식 체크
• IOMMU 활성화 확인: dmesg | grep -e IOMMU -e iommu
  → "iommu: Default domain type: Passthrough" 확인
• vfio 모듈 로드 확인: lsmod | grep vfio
• GPU 바인딩 확인: lspci -nnk -s 6e:00
  → "Kernel driver in use: vfio-pci" 확인
↓
❓ Kernel driver in use: vfio-pci ?
✅ YES
다음 단계 진행
❌ NO
설정 재확인 후 재부팅
↓ YES
STEP 6. VM에 GPU 패스스루 추가
• Display: none 설정
• Hardware → Add → PCI Device
• 0000:6e:00.0 (GPU) Primary GPU ✅ ROM-Bar ✅ PCI-Express ✅
• 0000:6e:00.1 (Audio) 추가
• VM 시작 후 드라이버 설치
↓
✅ GPU PassThrough 완료!
VM에서 AMD Radeon RX 7600 정상 인식 확인
⚠️ AMD Navi 33 (RX 7600) 주의사항
  • vendor-reset 미지원으로 GPU 리셋 버그 발생 가능
  • 고부하(벤치마크 등) 후 VM 재시작 시 GPU 인식 불가 현상
  • 해결방법: 완전 전원 차단(콜드 부팅) 후 재시작
  • 안정적인 패스스루: RX 6000 시리즈 또는 NVIDIA GPU 권장
